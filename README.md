# GoIT Markup Homework 04

## 📖 About the Project

This project was created as part of the **GoIT Fullstack Developer** course (HTML & CSS, Homework #4).

The goal of this assignment is to practice building a semantic web page using modern CSS techniques, including Flexbox, SVG icons, transitions, and hover effects.

## 🚀 Technologies Used

* HTML5
* CSS3
* Flexbox
* SVG Sprite
* CSS Transitions
* Git & GitHub

## ✨ Features

* Semantic HTML structure
* Responsive page layout
* Flexbox positioning
* SVG icons
* Smooth hover animations
* Clean and organized code
